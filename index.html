Link Graph for Jira Cloud

A type of Gantt chart that shows blocking relationships from left to right or from top to bottom.

View ALL connected issues, including the full epic and subtasks tree. The dependency map allows you to view, create, and delete links.

Start the application and view all related issues

Click the app button on the issue page

Add issues to the board to create a link

Click on the search bar and type the search words

Creation and deletion of links

Click on the first issue, then click on the second issue to create a "block" link. Alternatively, click the "custom link" button to open a modal window with link type options

Select the fields to be displayed

Click on the "Fields" menu item and choose the desired fields

Rotate the graph

Click on the "Rotation" menu item.

Choose a theme

Click the options menu and select the theme

Pan and zoom

Filter issues

Click on the "Filter" menu item and apply the desired filters
